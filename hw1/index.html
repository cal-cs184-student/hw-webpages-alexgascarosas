Overview

For homework 1 I built a software rasterizer pipeline for Scalable Vector Graphics (SVGs). I implemented... 1. a single-color triangle rasterization using edge functions, 2. a supersampling antialiasing using a per-subsample buffer and resolve step, 3. SVG transforms - translate/scale/rotate - in homogeneous coordinates, 4. barycentric coordinates for smooth interpolated vertex colors, 5. texture mapping with nearest & bilinear pixel sampling, & 6. a mipmapped level sampling - L_ZERO/L_NEAREST/L_LINEAR - including trilinear filtering.

The most interesting takeaway was that everything is built from the same triangle backbone. It is cool and interesting to see that everything in homework 1 was ultimatley made from multiple single triangles.

Task 1: Drawing Single-Color Triangles

Algorithm

To rasterize a triangle, I first transform its three vertices into screen space. Once I have the 2D coordinates, I computed the triangle’s axis-aligned bounding box by taking the minimum and maximum x and y values among each vertex. I then iterate over every pixel inside this bounding box. For each sample point, I determined whether it lies inside the triangle using edge function. Each triangle edge defines a line, and the interior of the triangle is the intersection of the three corresponding half-planes. If the sample lies on the correct side of all three edges, it is inside the triangle. For these samples inside the triangle, I finally computed barycentric coordinates to interpolate attributes such as color and depth.

Why algorithm is no worse than bounding-box sampling

The algorithm only checks samples inside the triangle’s bounding box. We let N be the number of samples inside the triangle's bounding box. Since the algorithm examines only samples within this bounding box and performs a constant-time inside-triangle test for each sample, this would take constant time. Runtime is O(N). This is no worse than an algorithm that checks every sample within the bounding box, since both approaches iterate over the same set of candidate samples and perform constant work per sample.
