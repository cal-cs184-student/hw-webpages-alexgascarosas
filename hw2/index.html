Overview

In this homework, we implemented several geometric modeling techniques related to Bezier curves, Bezier surfaces, and triangle meshes. Overall, the assignment focused on understanding how smooth curves and surfaces can be represented mathematically and how triangle meshes can be manipulated using the half-edge data structure.

In the first section, we implemented Bezier curve evaluation using de Casteljau’s algorithm. The idea behind the algorithm is to repeatedly linearly interpolate between control points using a parameter t. Each step creates a smaller set of intermediate points until only one point remains, which lies on the Bezier curve. We implemented the function that performs one step of this interpolation and then used it recursively to evaluate the curve.

In the first section, we implemented Bezier curve evaluation using de Casteljau’s algorithm. The idea behind the algorithm is to repeatedly linearly interpolate between control points using a parameter t. Each step creates a smaller set of intermediate points until only one point remains, which lies on the Bezier curve. We implemented the function that performs one step of this interpolation and then used it recursively to evaluate the curve.

Next, we extended this idea to Bezier surfaces. Instead of evaluating a single curve, the surface evaluation works by applying the same 1D algorithm in two directions. First, each row of control points is treated as a Bezier curve and evaluated using parameter u. This produces a set of points that form another curve, which is then evaluated using parameter v to get the final point on the surface.

In the second part of the homework, we worked with triangle meshes using the half-edge data structure. We implemented area-weighted vertex normals, which are used for smooth shading. To compute this, we iterated over all faces connected to a vertex, computed each face’s normal using a cross product, weighted it by the face area, and summed them together before normalizing the result.

After that, we implemented two mesh operations: edge flips and edge splits. Edge flipping changes the diagonal shared by two triangles without adding or removing elements, while edge splitting inserts a new vertex at the midpoint of an edge and creates additional edges and faces. These parts required carefully updating many pointers in the half-edge structure to make sure all vertices, edges, faces, and halfedges still pointed to the correct elements.

Finally, we implemented loop subdivision to upsample the mesh. This algorithm subdivides each triangle into four smaller triangles and updates vertex positions using weighted averages of neighboring vertices. I first computed the new positions for vertices and edge midpoints, then performed edge splits and flips to subdivide the mesh, and finally updated the vertex positions.

One thing we found interesting about homework 2 is how much detail is involved in working with meshes. The math concepts like interpolation and subdivision are fairly straightforward, but implementing them correctly with the half-edge data structure required careful debugging and attention to how everything is connected.
